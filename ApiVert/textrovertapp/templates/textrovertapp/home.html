API Overview

Welcome to the ApiVert API !

About ApiVert API

ApiVert API is designed to streamline the process of generating conversation templates for a variety of topics, making it a valuable resource for developers building chatbots, social media automation, and more.

Key Features:

  1. Topic-specific Templates:
    • Easily access a rich collection of pre-defined templates for common daily life topics.
    • Topics include emotions (anger, sadness, love), special occasions (birthday, wishes), and more.
  2. Integration with Chatbots:
    • Simplify the development of chatbots by integrating our API to provide diverse and realistic responses.
    • Enhance user engagement and interaction with dynamic and topic-relevant conversations.
  3. Social Media Automation:
    • Automate social media posts with personalized and expressive content using our template library.
    • Effortlessly create captivating messages for different scenarios without the need for extensive manual content creation.
  4. Developer-Friendly:
    • Built on Django Rest Framework for a robust and scalable solution.
    • Straightforward API endpoints for easy integration into your applications and services.

How to Get Started:

  1. Explore Topics:
    • Review available topics and templates in our documentation.
    • Choose topics that align with your application's needs.
  2. Integrate with Your Application:
    • Use simple API requests to fetch templates and integrate them seamlessly into your chatbot or automation workflows.

Usage

To interact with the ApiVert API, follow these steps:

1. Making Requests without API Key

You can make requests to the ApiVert API without an API key when using it locally. Here's an example using cURL:

            
                curl http://localhost:8000/all/

Replace `http://localhost:8000/api-endpoint/` with the actual endpoint of your ApiVert API hosted locally.

2. Making Requests using Python

If you prefer using Python, you can make requests using the `requests` library without an API key:

            
                import requests

                # Replace API_ENDPOINT with your actual API endpoint
                api_endpoint = "http://localhost:8000/api-endpoint/"

                response = requests.get(api_endpoint)

                # Process the API response
                print(response.json())

Replace `http://localhost:8000/api-endpoint/` with the actual endpoint of your ApiVert API and name of website hosted locally

Examples

You can get all JSON data by adding "/all" to the end of the actual website

Example Request - All Data

To retrieve information about all categories, make a request: 

                    
                        curl http://localhost:8000/all/

Example Requests - Specific Endpoints

To retrieve information about specific categories, make requests: 

                    
                        curl http://localhost:8000/birthday/
                        curl http://localhost:8000/sad/
                        curl http://localhost:8000/happy/
                        curl http://localhost:8000/fear/
                        curl http://localhost:8000/surprise/
                        curl http://localhost:8000/anger/
                        curl http://localhost:8000/help/
                        curl http://localhost:8000/love/

Example Response

The API will respond with a JSON object containing template details: 

                            
                                {
                                    "id": 1,
                                    "name": "birthday_1",
                                    "data": "Happy Birthday",
                                    "category": "Birthday"
                                }
